# Student Management System (Java)

## 📌 Description
A console-based application developed using Java to manage student records efficiently.

## 🚀 Features
- Add student details
- View all students
- Update student records
- Delete student records

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList

## 💡 Concepts Covered
- CRUD Operations
- Exception Handling (basic)
- User Input Handling

## ▶️ How to Run
1. Compile:
   javac StudentManagementSystem.java

2. Run:
   java StudentManagementSystem
